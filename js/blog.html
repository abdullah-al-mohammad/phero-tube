Discuss the scope of var, let, and const?

Answere: 1.var:- Variables declared with var can have a global or local scope. Global scope is for variables declared outside functions, while local scope is for variables declared inside functions. 2.let:- Variables declared with let can have a global, local, or block scope. Block scope is for variables declared in a block. A block in JavaScript involves opening and closing curly braces: 3:- Variables declared with const are similar to let in regards to scope. Such variables can have a global, local, or block scope.
